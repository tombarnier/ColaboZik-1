## Descriptionaaa

  Application mobile permettant de créer des playlists à plusieurs avec un système de vote pour retirer les musiques.
  
## Fonctionnalités

  - Gestion utilisateur
  - Création de groupe pour gérer une playlist
  - Gestion des votes
  - Lecture de playlistsaaaaaaaa
  
## Choix techniques

### Client

  - React Native
  - React Navigation
  - Redux
  - Socket.io
  - Styled Components
  - Native Base
  - React Native Youtube
